# General Coding Standards

As an AI agent working on this project, you must adhere to the following general coding standards:

## 1. Code Quality

- **Clean Code**: Write code that is easy to read and maintain.
- **DRY (Don't Repeat Yourself)**: Avoid code duplication. Extract common logic into reusable functions or components.
- **KISS (Keep It Simple, Stupid)**: Prefer simple solutions over complex ones.

## 2. Naming Conventions

- **Files & Directories**: Use `kebab-case` for all files and directories.
- **Variables & Functions**: Use `camelCase`.
- **Components**: Use `PascalCase`.
- **Constants**: Use `SCREAMING_SNAKE_CASE`.

## 3. Project Structure

- **`app/`**: Next.js App Router pages and layouts.
- **`components/`**: Reusable UI components.
- **`lib/`**: Utility functions and shared logic.
